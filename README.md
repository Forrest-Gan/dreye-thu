  * Macbook Pro 2017 4 Thurder 3 
  * Python 3.6
  * Tensorflow 2.0
  
## How to run
Just download the repo and start `predict.py`. Together with code, we release pre-trained weights and sample data from the DR(eye)VE dataset to test.
